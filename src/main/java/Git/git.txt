# git add fileName - Подготавливаем новое содержимое к коммиту. Эту команду нужно применять даже в том случае, если файл был просто изменён.
# git commit -m 'commitDescription' - Непосредственно добавление изменения в репозитории с описанием.
# git log - Просмотр истории коммитов.
@ git clone url - Клонирование существующего репозитория.
# remote - Удаленный репозиторий.
